How to deploy (for Aqsa)

  1. Duplicate the live adset Uniqu_cctv_residential_Pricing → name new duplicate Uniqu_cctv_residential_2026-05-creative-test.
  2. For each concept, create a new ad in the duplicated adset. Same lead form (1856073175022637), same GET_QUOTE CTA. UTM utm_content=2026-05-{concept-slug}.
  3. Upload all 3 sizes per concept — Meta auto-routes by placement when all aspect ratios are present.
  4. Run a 7-day budget split: 33/33/34 across the 3 new ads. Don't pause the current Ad 2 winner — let it compete.
  5. Read at day 7: leads-per-dollar against current Ad 2 baseline (A$7.71/lead). Pause anything underperforming by >30%.
What to watch. Account frequency is 2.88 (approaching fatigue). New creative is the right next move; refreshing the same hook isn't enough. CTR for the new pack should beat 2.41% baseline; cost/lead should beat A$12.61.
